# Tetris
Small Tetris game developed using python and pygame.

**Controls:**\
  Pause/unpause game - "P"\
  Move shape left - "left arrow key"\
  Move shape right - "right arrow key"\
  Rotate shape - "up arrow key"\
  Hard drop shape - "space bar"
